Desarrollar un sistema inteligente que apoye a INCOPESCA, MarViva, Guardacostas y pescadores artesanales a identificar especies 
marinas, predecir condiciones oceanográficas y recomendar zonas de pesca sostenible, usando modelos CNN, RNN/LSTM en Python.

## 📋 Qué herramientas de IA se utilizaron y para qué tareas específicas
- Google Gemini se utilizó para tener guía sobre la estructura de un proyecto profesional principalmente en Github y ramas, para consultas de errores y para entendimiento de liberias utilizadas en el apartado de materia vista en clase.
- Antigravity se utilizó para lectura del proyecto y validaciones y encontrar partes específicas de manera mas veloz, además de apoyo para crear apartados del EDA.
